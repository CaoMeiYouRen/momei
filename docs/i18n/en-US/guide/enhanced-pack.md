# Enhanced Pack

::: warning Translation Notice
The Enhanced Pack is currently in early development. This page is a brief translated introduction. For the latest and most complete information, please refer to the [authoritative Chinese version](../../../guide/enhanced-pack.md).
:::

**Let your technical content reach readers worldwide — automatically.**

The Enhanced Pack is an optional value-added service built on top of Momei's open-source core. It provides three advanced capabilities beyond the free tier:

- **Batch Translation Orchestration**: Translate multiple articles to multiple languages in a single batch, with terminology consistency checks and pre-publish comparison previews.
- **Multi-Platform Distribution Tracking**: Track distribution status across GitHub, Gitee, Dev.to, Hashnode, and more — with failure retry and consistency governance.
- **Unified Content Asset Convergence**: Cross-entry indexing and association for blogs, docs, and project READMEs in a single unified view.

The Enhanced Pack is still in development and validation. [Join the waitlist](https://momei.app/benefits) to be notified when it opens.

For detailed information on the Enhanced Pack's capabilities, the free-core vs enhanced-pack comparison, and FAQs, please see the [Chinese authoritative version](../../../guide/enhanced-pack.md).
