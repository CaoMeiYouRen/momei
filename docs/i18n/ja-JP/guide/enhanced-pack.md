# 拡張パック

::: warning 翻訳に関する注意
拡張パックは現在早期開発段階です。このページは簡易的な翻訳紹介です。最新かつ完全な情報は [簡体字中国語の正文版](../../../guide/enhanced-pack.md) をご参照ください。
:::

**あなたの技術コンテンツを世界中の読者に自動で届ける。**

拡張パックは Momei オープンソースコアの上に構築されるオプションの付加価値サービスです。無料コアに加えて、以下の 3 つの高度な機能を提供します：

- **一括翻訳オーケストレーション**：複数の記事を複数言語へ一括翻訳し、用語整合チェックと公開前の比較プレビューを提供。
- **マルチプラットフォーム配信追跡**：GitHub、Gitee、Dev.to、Hashnode などへの配信状態を統合追跡し、失敗リトライと整合性ガバナンスを提供。
- **コンテンツアセット統合管理**：ブログ、ドキュメント、プロジェクト README など、複数エントリーのコンテンツを統一インデックスで関連付け。

拡張パックは現在開発・検証段階です。[ウェイトリストに登録](https://momei.app/benefits) して、公開時のお知らせをお受け取りください。

詳細な機能説明、無料コア vs 拡張パックの比較、よくある質問については、[簡体字中国語の正文版](../../../guide/enhanced-pack.md) をご参照ください。
