# UI 설계

::: warning 번역 안내
이 페이지는 [중국어 원문](../../../design/ui.md)을 기준으로 정리했습니다. 차이가 있을 경우 원문을 우선합니다.
:::

모메이의 UI는 동양적 정서와 현대적인 웹 경험을 함께 추구합니다. 공개 블로그와 관리자 화면 모두에서 읽기 쉬움, 몰입감, 운영 효율을 동시에 달성하는 것이 목표입니다.

## 1. 디자인 철학

- 간결함: 콘텐츠를 주인공으로 둡니다.
- 현대성: 과하지 않은 애니메이션과 상태 피드백을 사용합니다.
- 몰입감: 읽기와 쓰기 흐름을 방해하지 않습니다.
- 프라이버시 / 저마찰: 필요한 순간에만 인증 강도를 높입니다.

## 2. 시각 시스템

- PrimeVue Aura 기반 커스터마이징
- 라이트 / 다크 / 시스템 모드 지원
- 핵심 색상과 표면색, 강조색의 관리자 구성 가능성
- 코드용 모노스페이스 폰트와 읽기 친화적 본문 타이포그래피

## 3. 레이아웃과 컴포넌트

- 버튼, 카드, 입력 컴포넌트의 일관된 상태 표현
- 장문 읽기에 적합한 콘텐츠 폭 제어
- 관리자용 공통 헤더, 확인 다이얼로그, 표, 번역 편집기 추상화

## 4. 반응형과 상호작용

- Mobile First
- 모바일 내비게이션과 큰 터치 영역 보장
- 코드 블록, 표, 이미지의 overflow 처리
- 페이지 전환은 부드럽고 짧게, 로딩은 skeleton 우선

## 5. 유지보수 원칙

- 기존 디자인 언어를 깨는 새로운 스타일 도입을 피합니다.
- 중요한 UI 변경은 실제 렌더링 검증을 함께 수행합니다.
