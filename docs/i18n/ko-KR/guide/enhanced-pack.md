# 확장 팩

::: warning 번역 안내
확장 팩은 현재 초기 개발 단계입니다. 이 페이지는 간략한 번역 소개입니다. 최신의 완전한 정보는 [중국어 간체 권위 버전](../../../guide/enhanced-pack.md)을 참조하세요.
:::

**기술 콘텐츠를 전 세계 독자에게 자동으로 전달하세요.**

확장 팩은 모메이 오픈소스 코어 위에 선택적으로 추가되는 부가가치 서비스입니다. 무료 코어 외에 세 가지 고급 기능을 제공합니다:

- **일괄 번역 오케스트레이션**: 여러 게시글을 여러 언어로 한 번에 번역하고, 용어 일관성 검사 및 게시 전 비교 미리보기를 제공합니다.
- **멀티 플랫폼 배포 추적**: GitHub, Gitee, Dev.to, Hashnode 등으로의 배포 상태를 통합 추적하며, 실패 재시도와 일관성 거버넌스를 지원합니다.
- **콘텐츠 자산 통합 관리**: 블로그, 문서, 프로젝트 README 등 여러 진입점의 콘텐츠를 통합 인덱싱하여 연관 관계를 보여줍니다.

확장 팩은 현재 개발 및 검증 단계에 있습니다. [대기자 명단에 등록](https://momei.app/benefits)하시면 오픈 시 알림을 받으실 수 있습니다.

자세한 기능 설명, 무료 코어 vs 확장 팩 비교, 자주 묻는 질문은 [중국어 간체 권위 버전](../../../guide/enhanced-pack.md)을 참조하세요.
