# 增強包

::: warning 翻譯說明
增強包目前處於早期開發階段。本頁為簡介翻譯，最新最完整的資訊請以 [簡體中文權威版本](../../../guide/enhanced-pack.md) 為準。

**讓你的技術內容自動觸及全球讀者。**

增強包是墨梅開源部落格之上的可選增值服務，在免費核心之外提供三項進階能力：

- **批量翻譯編排**：多篇文章一次批量翻譯到多種語言，包含術語一致性檢查與發佈前比對預覽。
- **多平台分發追蹤**：GitHub、Gitee、Dev.to、Hashnode 等平台的分發狀態統一追蹤，支援失敗重試與一致性治理。
- **內容資產統一收斂**：部落格、文件、專案 README 等跨入口內容資產的統一索引與關聯展示。

增強包目前仍在開發與驗證階段。[加入候補名單](https://momei.app/benefits)，我們會在開放時第一時間通知你。

詳細的能力說明、免費核心 vs 增強包對照與常見問題，請查閱 [簡體中文權威版本](../../../guide/enhanced-pack.md)。
