# Momei 개발 표준

::: warning 번역 안내
이 페이지는 [중국어 원문](../../standards/development.md)과 [영문판](../../en-US/standards/development.md)을 바탕으로 정리했습니다. 차이가 있을 경우 중국어 원문을 우선합니다.
:::

## 1. 범위

이 표준은 디렉터리 구조, 코드 스타일, Vue / TypeScript 작성 방식, i18n, 스타일, 문서 동기화 요구사항을 다룹니다.

## 2. 일상 개발 규칙

- 변경은 작고 명확하게 유지합니다.
- 기존 구조와 명명 방식을 우선 따릅니다.
- 새 기능은 lint, typecheck, 테스트, 문서에 포함되어야 합니다.
