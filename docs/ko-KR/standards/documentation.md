## 1. 문서 구조

- `guide/`: 사용자 및 개발 안내

## 2. 유지보수 요구사항

- 각 문서는 하나의 H1만 사용합니다.
- 새 페이지를 추가하면 nav/sidebar를 함께 갱신합니다.
- 번역 문서는 `source_branch` 와 `last_sync` 를 포함합니다.
