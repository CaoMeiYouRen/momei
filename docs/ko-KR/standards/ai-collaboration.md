# AI 협업 표준

::: warning 번역 안내
이 페이지는 [중국어 원문](../../standards/ai-collaboration.md)과 [영문판](../../en-US/standards/ai-collaboration.md)을 바탕으로 정리했습니다. 차이가 있을 경우 중국어 원문을 우선합니다.
:::

## 1. 기본 원칙

- AI는 먼저 맥락을 읽고 그다음에 변경을 제안하거나 구현합니다.
- 중대한 변경은 테스트, 문서, 위험 설명을 함께 수반해야 합니다.
- 검증하지 않은 동작을 사실처럼 기술하지 않습니다.

## 2. 프로젝트 내 적용 방식

- PDTFC+ 순환을 준수합니다.
- AGENTS, CLAUDE, skill 문서, 표준 문서를 함께 따릅니다.
- 문서 사이트 구조가 바뀌면 VitePress nav/sidebar를 함께 갱신합니다.
