# 변수 및 설정 매핑

::: warning 번역 안내
이 페이지는 [중국어 원문](../../guide/variables.md)과 [영문판](../../en-US/guide/variables.md)을 바탕으로 정리했습니다. 차이가 있을 경우 중국어 원문을 우선합니다.
:::

## 1. 설정 우선순위

시스템 설정은 일반적으로 다음 순서를 따릅니다.

1. 환경 변수
2. 데이터베이스 설정
3. 시스템 기본값

## 2. 우선 확인할 변수 그룹

- 인증: `AUTH_SECRET` 및 로그인 관련 설정
- AI: 제공자, 모델, 비용, 쿼터 정책
- 저장소: 로컬, S3, Cloudflare R2, Vercel Blob
- 이메일 및 알림: SMTP, Web Push, 운영 알림

## 3. 사용 팁

- 프로덕션의 민감 정보는 환경 변수로 관리합니다.
- 필드가 ENV에 의해 잠겨 있는지 먼저 확인합니다.
- 상세 매핑은 [중국어 원문](../../guide/variables.md)을 참고합니다.
