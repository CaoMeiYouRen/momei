# 기능 개요

::: warning 번역 안내
이 페이지는 [중국어 원문](../../guide/features.md)과 [영문판](../../en-US/guide/features.md)을 바탕으로 정리했습니다. 차이가 있을 경우 중국어 원문을 우선합니다.
:::

## 1. 핵심 기능

- AI 창작 보조: 제목, 요약, 번역, 음성 원고, 표지 이미지 생성
- 네이티브 국제화: locale registry, 언어 전환, 다국어 SEO, 번역 거버넌스
- 발행과 알림: 게시글, 이메일, Memos, 알림 센터, 운영 알림 체계
- 운영형 관리자 화면: 테마, 상업화, 친구 링크, AI 비용 및 쿼터 관리

## 2. 이런 경우에 적합합니다

- Nuxt 기반의 현대적인 기술 블로그를 구축하고 싶을 때
- AI와 다국어 콘텐츠 운영을 하나의 워크플로로 관리하고 싶을 때
- 발행, 알림, 설정, 운영 기능을 하나의 백오피스로 묶고 싶을 때
