# 기능 소개

::: warning 번역 안내
이 페이지는 [중국어 원문](../../guide/features.md)을 기준으로 정리했습니다. 차이가 있을 경우 원문을 우선합니다.
:::

모메이는 단순히 AI를 몇 가지 버튼 형태로 붙여 놓은 블로그가 아닙니다. 글쓰기, 번역, 배포, 구독, 알림, 운영 관리까지를 하나의 흐름으로 연결해 주는 **AI 기반 네이티브 국제화 블로그 플랫폼**입니다.

## 1. AI 창작 보조

- 본문을 기반으로 한 제목 제안
- SEO 설명에 활용할 수 있는 자동 요약
- 태그 추천과 구조화된 글쓰기 보조
- SEO 친화적 Slug 생성
- Markdown 형식을 유지하는 전체 번역
- AI 표지 이미지 생성
- 음성 입력을 초안으로 바꿔 주는 ASR 지원
- 흩어진 아이디어를 수집하는 영감 캡처 흐름

## 2. 네이티브 국제화

- 번역 클러스터 기반의 다국어 콘텐츠 관리
- 언어 전략을 고려한 라우팅과 `hreflang` 출력
- 언어 전환 시 대응되는 글 버전으로 이동하는 정밀 점프
- 분류와 태그까지 포함하는 다국어 메타데이터 일관성

## 3. 구독과 배포

- 전체 사이트, 분류, 태그별 RSS / Atom / JSON Feed
- 현재 페이지에 맞는 피드 자동 발견
- 분류 / 태그별 이메일 구독 기반 확장 가능성
- WechatSync 등 외부 배포 흐름 연동

## 4. 검색과 읽기 경험

- 로컬 인덱스 기반 전체 검색
- `Ctrl+K` / `Cmd+K` 단축키로 검색 호출
- 키워드, 분류, 태그, 언어 필터링
- KaTeX 수식 지원
- 자동 TOC 생성
- 읽기 모드, 글꼴 크기, 행간, 폭 조절
- 예상 읽기 시간 계산
- 저작권 프로토콜 표시와 다크 모드 지원

## 5. 운영과 거버넌스

- 다양한 CAPTCHA 제공자 지원
- 댓글 심사 체계
- Markdown 기반 약관 / 개인정보 처리방침 관리
- 다국어 법률 문서 지원
- 설정 센터, 환경 변수 잠금, 배포 가이드의 일관된 연결

## 6. 개발자 경험과 자동화

- 설치 마법사
- Hexo 등에서 가져올 수 있는 Momei CLI
- 외부 발행용 API Key 기반 인터페이스
- 메모리 DB 기반 Demo 모드
- Memos 동기화 및 예약 작업 엔진
