# AI 주도 개발 가이드

::: warning 번역 안내
이 페이지는 [중국어 원문](../../guide/ai-development.md)과 [영문판](../../en-US/guide/ai-development.md)을 바탕으로 정리했습니다. 차이가 있을 경우 중국어 원문을 우선합니다.
:::

## 1. 핵심 워크플로

모메이는 PDTFC+ 리듬을 사용해 계획, 구현, 검증, 피드백, 문서 동기화를 하나의 반복으로 관리합니다.

## 2. AI 협업 시 요구사항

- 먼저 문서와 기존 구조를 읽고 나서 수정합니다.
- 구현 후에는 테스트, 문서, todo를 함께 갱신합니다.
- 보안, 권한, 설정, 다국어가 포함되면 관련 표준 문서를 함께 확인합니다.
