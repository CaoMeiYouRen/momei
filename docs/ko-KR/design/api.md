# API 설계

::: warning 번역 안내
이 페이지는 [중국어 원문](../../design/api.md)과 [영문판](../../en-US/design/api.md)을 바탕으로 정리했습니다. 차이가 있을 경우 중국어 원문을 우선합니다.

## 1. 설계 초점

- 공개 API와 관리자 API를 분리합니다.
- 공통 데이터 구조, 오류 응답, 검증 방식의 일관성을 유지합니다.
- 권한, 설정, 알림과 같은 시스템 기능과 자연스럽게 연결되어야 합니다.
