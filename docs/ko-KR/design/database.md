# 데이터베이스 설계

::: warning 번역 안내
이 페이지는 [중국어 원문](../../design/database.md)과 [영문판](../../en-US/design/database.md)을 바탕으로 정리했습니다. 차이가 있을 경우 중국어 원문을 우선합니다.
:::

## 1. 포함 범위

- 핵심 엔티티와 관계
- 게시글, 사용자, 분류, 태그, 알림, 설정 데이터 모델
- 저장소 백엔드와 확장 필드 전략

## 2. 사용 팁

- 데이터 구조 변경 전 API와 엔티티 영향을 함께 확인합니다.
- 큰 모델 변경은 설계 문서와 테스트를 함께 갱신합니다.
