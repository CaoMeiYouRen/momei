# UI 설계

::: warning 번역 안내
이 페이지는 [중국어 원문](../../design/ui.md)과 [영문판](../../en-US/design/ui.md)을 바탕으로 정리했습니다. 차이가 있을 경우 중국어 원문을 우선합니다.
:::

## 1. 디자인 방향

- 공개 블로그와 관리자 화면 모두 일관된 브랜드 감각을 유지합니다.
- 데스크톱과 모바일 경험을 함께 고려합니다.
- 테마, 읽기 모드, 상호작용 요소는 가독성을 최우선으로 둡니다.

## 2. 주요 관심사

- 타이포그래피와 색상 체계
- 상태, 상호작용, 피드백
- 다국어와 테마 전환 시의 시각적 일관성
