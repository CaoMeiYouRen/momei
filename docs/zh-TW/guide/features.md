# 功能特色

::: warning 翻譯說明
本頁對應 [中文原文](../../guide/features.md) 與 [英文版本](../../en-US/guide/features.md)。若內容有差異，請以中文原文為準。

## 1. 核心能力

- AI 創作輔助：標題、摘要、翻譯、語音稿與封面圖生成。
- 原生國際化：內建 locale registry、多語言切換、多語言 SEO 與語系治理。
- 發佈與通知：文章、郵件、Memos、通知中心與站務提醒鏈路。
- 可營運後台：主題、商業化、友站連結、AI 成本與額度治理。

## 2. 適合的使用情境

- 想要用 Nuxt 建立現代化技術部落格。
- 需要 AI 協作寫作與多語言內容管理。
- 希望把設定、通知、發佈與營運能力集中在同一後台。
