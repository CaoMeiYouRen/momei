# 功能特色

::: warning 翻譯說明
本頁對應 [中文原文](../../guide/features.md)。若內容有差異，請以中文原文為準。
:::

墨梅是一套 **AI 驅動、原生國際化** 的現代化部落格平台。它不只是把 AI 加到內容編輯器中，而是把創作、翻譯、營運、通知與多語系發佈串成一整套工作流，幫助技術創作者更自然地面向全球讀者。

## 1. AI 創作助手

墨梅內建深度整合的 AI 工作流，支援 OpenAI、Anthropic、DeepSeek、豆包等供應商，能力覆蓋從靈感到發佈的整個流程：

- **智慧標題建議**：依據文章正文自動生成多組更具吸引力且兼顧 SEO 的標題。
- **自動摘要**：一鍵提取文章核心內容，快速產生摘要與 SEO 描述。
- **智慧標籤推薦**：根據內容語意推薦合適的分類與標籤。
- **腳手架式寫作助手**：由 AI 協助搭建結構、提煉觀點，而不是只輸出一段成品文字。
- **SEO Slug 生成**：自動產出簡潔、可讀性高的 URL 別名。
- **全文翻譯**：保留 Markdown、程式碼區塊與公式格式，支援多語內容同步生成。
- **AI 配圖生成**：可透過 DALL-E 3、豆包 Seedream、SiliconFlow 等能力產生文章封面圖。
- **語音轉文字（ASR）**：支援 Whisper、SenseVoice 等方案，適合快速整理語音靈感。
- **靈感捕捉引擎**：讓碎片化想法先被收集，再逐步整理成正式文章。

## 2. 深度國際化

墨梅把國際化視為內容系統的一級能力，而不是只翻譯幾個 UI 字串：

- **翻譯簇管理**：同一篇內容的多語版本可在後台聚合管理。
- **智慧路由**：支援語言前綴、預設語言無前綴等策略，並可自動輸出 `hreflang`。
- **語言切換精準跳轉**：讀者切換語言時，系統會嘗試跳到對應文章的翻譯版本，而不是直接回首頁。
- **多語元資料隔離**：分類、標籤與公開內容都能依語言維持一致的上下文。

## 3. 全方位訂閱與分發

墨梅將訂閱能力視為內容營運的重要基礎設施：

- **RSS / Atom / JSON Feed**：除了全站訂閱，分類與標籤也能各自輸出獨立訂閱源。
- **智慧 RSS 發現**：讓瀏覽器與閱讀器自動找到當前頁面的最佳訂閱入口。
- **郵件訂閱**：支援依分類與標籤進行較精準的內容推送。
- **一鍵同步**：整合 WechatSync 等能力，將內容發佈到更多平台。

## 4. 搜尋與閱讀體驗

- **本機全文檢索**：依靠資料庫索引與搜尋 API 提供快速回應。
- **快捷鍵喚起**：可透過 `Ctrl+K` 或 `Cmd+K` 快速開啟搜尋對話框。
- **多維篩選**：支援關鍵字、分類、標籤與語言條件的即時過濾。
- **數學公式支援**：原生支援 KaTeX，適合技術文章與教學內容。
- **自動目錄**：依 Markdown 標題自動生成 TOC。
- **沉浸式閱讀模式**：提供字級、行高、寬度與配色的調整能力。
- **閱讀時間預估**：自動估算字數與閱讀時長。
- **版權協議展示**：支援 CC BY-NC-SA 等常見授權協議。
- **暗色模式與效能優化**：兼顧外觀一致性與載入體驗。

## 5. 安全、合規與營運治理

- **多供應商驗證碼**：支援 Cloudflare Turnstile、Google reCAPTCHA、hCaptcha 與 CaptchaFox。
- **評論審核機制**：控制垃圾內容與互動品質。
- **Markdown 法律協議管理**：可在後台維護使用者協議與隱私政策。
- **多語法律文本**：不同語系可使用獨立版本。
- **系統治理能力**：設定中心、環境變數鎖定、審計資訊與部署說明彼此打通。

## 6. 開發者體驗與遷移能力

- **安裝精靈**：協助完成首輪部署、資料庫與基礎設定。
- **Momei CLI**：支援從 Hexo 等系統將 Markdown + Front-matter 內容匯入。
- **外部發佈 API**：可透過 API Key 與外部自動化流程整合。
- **Demo 模式**：使用記憶體資料庫快速展示產品能力，適合試用與演示。

## 7. 碎片內容與自動化任務

- **Memos 同步**：將日常碎片想法同步到部落格或動態頁面。
- **定時任務引擎**：支援文章定時發佈、快取清理、外部資料同步等能力。
- **安全驗證機制**：可透過 `CRON_SECRET`、`WEBHOOK_SECRET`、`TASKS_TOKEN` 等方式保護任務端點。
