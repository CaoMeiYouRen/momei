# UI 設計

::: warning 翻譯說明
本頁對應 [中文原文](../../design/ui.md) 與 [英文版本](../../en-US/design/ui.md)。若內容有差異，請以中文原文為準。
:::

## 1. 設計方向

- 部落格公開頁與管理端需要維持一致品牌感。
- 兼顧桌面與行動端體驗。
- 主題、閱讀模式與互動元件要以可讀性優先。

## 2. 關注點

- 字體、色彩與版面階層
- 元件狀態與互動回饋
- 主題切換與多語言下的視覺一致性
