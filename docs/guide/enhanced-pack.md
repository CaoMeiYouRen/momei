# 多语言内容资产化增强包 (Enhanced Pack)

**让你的技术内容自动触达全球读者。**

增强包是墨梅开源博客之上的可选增值服务，在开源免费核心之外提供 **批量翻译编排**、**多平台分发追踪** 与 **跨入口内容资产统一管理** 三项核心能力。

## 1. 定位与主卖点

墨梅开源版已经覆盖了单篇文章的 AI 辅助创作、原生多语言 UI、RSS / Sitemap 等基础分发能力。但当你的内容规模增长到 **数十篇文章 × 多种语言 × 多个分发平台** 时，逐篇手动翻译、逐个平台追踪状态的成本会快速上升。

增强包的核心价值是：**把你从"单篇操作"升级为"资产化运营"**。

| 层级 | 能力范围 | 交付方式 |
|------|----------|----------|
| **开源免费核心** | Markdown 编辑器、AI 建议标题 / 摘要 / 标签、单篇文章 AI 翻译、多语言 UI（5 语）、RSS / Sitemap / Feed 分发、Demo 模式 | 开源仓库，自部署 |
| **增强包** | 批量翻译编排（多篇 → 多语）、术语一致性检查、翻译后预览与发布前检查、多平台分发状态追踪与失败重试、跨入口内容资产统一收敛 | 独立说明页 + 候补名单 |

::: warning 边界说明
增强包提供的编排、追踪、收敛能力在免费核心中不存在。我们不会把现有免费能力加锁伪装成增强包。
:::

## 2. 增强能力详解

### 2.1 批量翻译编排

你的博客积累了数十篇文章，希望全部翻译到日语、韩语和英语。开通增强包后：

- **批量调度**：一次性选择多篇文章 → 多个目标语言，系统自动分派翻译任务。
- **术语一致性检查**：翻译过程中自动检测跨文章的术语翻译是否一致，避免同一概念在不同文章中出现不同译法。
- **预览与比对**：翻译完成后的草稿支持侧边栏预览与原文对照，确认后再统一发布，不需要逐篇点开后台核对。

*对比免费核心：免费版仅支持单篇文章的逐篇手动翻译。*

### 2.2 多平台分发追踪

文章在墨梅发布后，你可能还需要同步到 GitHub、Gitee、Dev.to、Hashnode 等外部平台。增强包帮你：

- **统一分发面板**：在一个界面查看所有跨平台分发任务的状态。
- **失败重试与状态回写**：当某个平台同步失败时，系统会记录失败原因并支持人工重试。
- **一致性治理**：确保各平台的发布时间、内容版本等保持一致。

*对比免费核心：免费版支持基础 RSS / Sitemap 等协议级分发，但不支持外部分发状态追踪与重试。*

### 2.3 跨入口内容资产统一收敛

博客、项目 README、文档站……你的内容可能分散在多个入口。增强包提供：

- **统一索引**：将跨入口的内容资产纳入统一索引，便于全局搜索与关联。
- **跨入口展示编排**：在不同页面之间智能关联相关内容，增强读者的内容发现路径。

*对比免费核心：免费版各入口（博客、文档等）独立运营，缺少跨入口的统一视图与关联能力。*

## 4. 常见问题

### 增强包会影响我现有的博客吗？

不会。增强包是墨梅开源核心之上的可选增值服务，不安装增强包时，你的博客完全照常运行。

### 我需要把博客迁移到别的地方吗？

不需要。增强包是对你已有墨梅部署的能力延伸，不需要数据迁移或平台切换。
