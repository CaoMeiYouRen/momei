# 功能特性 (Features)

墨梅 (Momei) 是一款 **AI 驱动、原生国际化** 的现代化博客平台。它不仅集成了深度 AI 创作助手，更通过底层的国际化设计，实现“AI 赋能，全球创作”，帮助技术创作者无缝连接全球读者。

## 1. AI 创作助手 (AI Power)

墨梅内置了深度集成的 AI 助手（支持 OpenAI, Anthropic, DeepSeek, 豆包等），覆盖了从灵感到发布的整个流程：

-   **智能标题建议**: 基于文章正文，自动生成多个具有吸引力且 SEO 友好的标题。
-   **自动化摘要 (Abstract)**: 一键提取文章精华，自动填充 SEO 描述。
-   **智能打标**: 自动分析内容并推荐合适的标签。
-   **脚手架式创作助手**: AI 提供逻辑提纲与论点引导，辅助创作者快速搭建文章框架。
-   **SEO Slug 生成**: 基于标题生成简洁、可读性强的 URL 别名。
-   **一键全文翻译**: 完美保留 Markdown 格式（含代码块、公式），支持将内容、分类、标签同步翻译为多种语言。
-   **AI 配图生成**: 集成 DALL-E 3、豆包 Seedream、SiliconFlow 等，为文章一键生成高质量封面图。
-   **语音转文字 (ASR)**: 支持通过 OpenAI Whisper、SenseVoice (SiliconFlow) 等将语音灵感快速转化为文字草稿。
-   **灵感捕捉引擎**: 实现碎片化想法的快速收集与智能分类，随时转化为正式文章。

## 2. 深度国际化 (I18n Anywhere)

我们认为国际化不应只是 UI 的翻译，而是内容的连接：

-   **翻译簇管理**: 在管理后台，同源的不同语言版本会被聚合并显示。你可以轻松在一个界面内管理文章的各语言版本。
-   **智能路由**: 支持多种 URL 策略（前缀、默认语言无前缀等），自动为页面生成 `hreflang` 标签。
-   **语言关联跳转**: 当读者在阅读某一篇文章时切换语言，墨梅会智能跳转到该文章对应的翻译版本，而不是简单的回到首页。
-   **内容隔离**: 分类和标签系统同样支持多语言，确保读者在特定语言环境下看到的是一致的元数据。

## 3. 全方位订阅系统 (Subscription Hub)

墨梅建立了一套强大的“订阅枢纽”，确保你的读者不会错过任何更新：

-   **多维度 RSS/Atom/JSON Feed**: 除了全局订阅地址，还为每个分类、每个标签都生成了独立的订阅源。支持 Atom 1.0, JSON Feed 1.1 等多种协议。
-   **智能 RSS 发现**: 浏览器和阅读器会自动探测到当前页面的最优订阅链接。
-   **进阶邮件订阅 (Newsletter)**: 访客可按需订阅特定分类或标签。通过内置的营销活动（Marketing Campaign）引擎，实现精准的内容推送。
-   **一键同步 (WechatSync)**: 集成微信号同步助手 SDK，支持将文章快速同步至微信公众号等平台。

## 4. 极致搜索体验 (Advanced Search)

-   **本地全文检索**: 基于数据库索引优化，响应速度极快（通常 < 100ms）。
-   **快速唤起**: 全站支持 `Ctrl+K` (或 `Cmd+K`) 快捷键弹出搜索对话框。
-   **多维过滤**: 支持按关键词、分类、标签和语言进行实时过滤。

## 5. 阅读体验与 SEO

-   **数学公式支持**: 原生支持 $LaTeX$ 数学公式渲染 (KaTeX)，支持行内与块级公式。
-   **自动生成 TOC**: 根据 Markdown 标题自动生成侧边栏目录。
-   **沉浸式阅读模式**: 隐藏所有视觉噪音，提供自定义字号、行高、页宽和多种护眼配色主题。
-   **全链路性能优化**: 核心页面 Lighthouse 指标全面达标 (>= 90)，提供极致的加载速度。
-   **阅读时间预估**: 自动计算字数并估算阅读时长。
-   **版权协议系统**: 支持 CC BY-NC-SA 4.0 等常见协议，并在文章底部显著展示。
-   **暗色模式支持**: 完美匹配系统的深色/浅色偏好，且在切换时无闪烁。
-   **自动 Sitemap**: 实时生成 `sitemap.xml`，自动通知搜索引擎更新。

## 6. 安全与防骚扰 (Security & Protection)

-   **多厂商验证码集成**: 支持 Cloudflare Turnstile, Google reCAPTCHA, hCaptcha 和 CaptchaFox，有效防止暴力破解和自动化注册攻击。
-   **评论审核机制**: 提供原生评论系统并支持开启分级审核，防止垃圾内容滋生。

## 7. 法律协议与合规 (Compliance)

-   **Markdown 协议管理**: 支持在后台以 Markdown 格式管理“用户协议”和“隐私政策”。
-   **多语言版本**: 支持为不同语言设置独立的法律文本。
-   **内置模板**: 提供默认的法律协议示例模板，支持一键复制粘贴。详情请参阅 [法律协议模板](./legal-templates.md)。

## 8. 开发者、迁移与演示模式

-   **安装向导**: 全新安装时提供图形化交互引导（Installation Wizard），轻松完成数据库与基础变量配置。
-   **极速迁移工具 (Momei CLI)**: 专为开发者设计的命令行工具，支持从 Hexo 等系统通过 Markdown (Front-matter) 一键导入内容。
-   **外部发布 API**: 通过 API KEY，你可以将墨梅集成到你的自动化流中（如从 Obsidian 一键发布）。
-   **Demo 模式**: 开启 `DEMO_MODE` 后，网站将使用内存数据库运行，并拦截所有持久化修改，非常适合作为产品演示。

## 9. Memos 碎片生活同步

-   **无缝集成**: 支持连接到你的 Memos (v1 API) 实例，将碎片化的想法同步至博客。
-   **流式呈现**: 在前台页面实时展示最新的 Memos 动态，打造个人“生活流”。

## 10. 定时任务与自动化

-   **内置调度引擎**: 基于 `TASKS_TOKEN` 安全验证，支持通过 Webhook 或系统 Cron 触发任务。
-   **自动化管理**: 支持定时发布文章、清理缓存、同步外部数据（如 Memos）等后台任务。
