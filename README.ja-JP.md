> **Momei - AI 駆動・ネイティブ国際化対応の開発者向けブログプラットフォーム**
>
> **AI Powered, Global Creation.**

## 📖 概要

Momei は **Nuxt** ベースのモダンなブログプラットフォームです。AI と深い国際化設計を組み合わせ、技術開発者やグローバルなクリエイター向けに、より効率的で知的な執筆体験を提供します。スマート翻訳、自動要約、多言語ルーティング管理など、グローバルな読者にリーチするための機能を提供します。

## ✨ 主な特徴

- **AI 駆動**: AI アシスタントとの深い統合、自動化翻訳、スマートタイトル、要約生成などをサポートし、執筆効率を大幅に向上。
- **マルチモーダルコンテンツワークフロー**: AI 画像、ASR、再利用可能な音声入力、Memos / WechatSync 手動配信、定期タスク自動化をサポートし、アイデア収集から公開までの完全なフローをカバー。
- **ネイティブ i18n**: UI からコンテンツ管理までの深い統合された多言語サポート。
- **モダンな技術スタック**: Nuxt (Vue 3 + TypeScript) ベース、SSG/SSR 混合レンダリングをサポート。
- **スムーズな移行**: カスタム URL Slug (パスエイリアス) をサポートし、古いブログからの SEO 損失ゼロを保証。
- **Markdown 創作**: リアルタイムプレビューと画像ドラッグ＆ドロップアップロードをサポートする効率的 Markdown エディタ。
- **コンテンツ構成とブランドセマンティクス**: ホーム画面の「最新記事 + 人気記事」双ブロック、記事ピン留め設定済み。
- **マルチレイヤーサブスクリプション**: グローバル、分類、タグ別の多次元 RSS 購読、多言語検出対応。
- **構成可能なシステムガバナンス**: 設定センター、環境変数ロック、監査ログ、展開ガイドが統合済み。
- **クラウドリソース配信**: S3 / R2 ダイレクトアップロード認可をサポート。

## 🛠️ 技術スタック

- **コアフレームワーク**: [Nuxt](https://nuxt.com/)

## 📂 プロジェクト構造

- `components/`: 再利用可能な Vue コンポーネント
- `pages/`: ファイルベースのページルート
- `layouts/`: ページレイアウト
- `server/`: Nitro サーバーサイド API インターフェースとエンティティ
- `database/`: データベース初期化スクリプトとリソース
- `i18n/`: 国際化言語設定ファイル
- `utils/`: 共有ユーティリティ関数と汎用ロジック
- `styles/`: グローバル SCSS スタイル定義
- `types/`: TypeScript インターフェースと型定義
- `docs/`: プロジェクト詳細ドキュメントと規範説明
- `packages/cli/`: Hexo 移行ツール CLI (独立プロジェクト)

## ☁️ 展開説明

### サポート状況

Vercel、Netlify、Docker または自己ホスト Node 環境での展開を推奨します。Cloudflare 連携が必要な場合は、現時点では R2 オブジェクトストレージや Scheduled Events などの周辺機能に限定して利用するのが安全です。TypeORM と Node ランタイム依存の制限により、現在のバージョンは Cloudflare Pages / Workers へのアプリケーション本体展開をサポートしていません。

現在のバージョンの展開設定は環境変数が主です。展開ガイドと環境設定を読んでから順に展開してください。

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FCaoMeiYouRen%2Fmomei)

### データベースサポート

Momei は以下のデータベースをネイティブサポート:

- **SQLite**: デフォルトオプション。サーバー設定が不要で、個人ブログに最適です。`DATABASE_URL=sqlite://database/momei.sqlite` の設定を推奨します。
- **MySQL / PostgreSQL**: より高いデータ管理要件を持つユーザー向け、`DATABASE_URL` プロトコルヘッダーで自動推測。
- **Cloudflare D1**: 計画中。これは現時点で Cloudflare ランタイム全体を正式サポートしていることを意味しません。現段階では外部データベースを使い、アプリ本体は Vercel、Docker、または自己ホスト Node 環境に配置してください。

詳細については [展開ガイド](https://docs.momei.app/ja-JP/guide/deploy) を参照。

## 🔄 Hexo 移行ツール

Momei は Hexo ブログシステムから記事を迅速に移行するための独立 CLI ツールを提供。

### 機能特性

- ✅ ディレクトリ内のすべての Markdown ファイルを再帰的にスキャン
- ✅ Hexo Front-matter (YAML 形式) を正確に解析
- ✅ 公開日時、分類、タグなどのメタデータを保持
- ✅ API Key によるバッチインポートをサポート
- ✅ 並行インポートをサポートして効率を向上
- ✅ Dry Run モードでプレビューをサポート

### クイック使用

```bash
# CLI ディレクトリに入る
cd packages/cli

# 依存関係をインストール
pnpm install

# ツールをビルド
pnpm build

# プレビューインポート（実際のインポートなし）
pnpm start import ./hexo-blog/source/_posts --dry-run --verbose

# 正式インポート
pnpm start import ./hexo-blog/source/_posts \
  --api-url https://your-blog.com \
  --api-key your-api-key-here
```

詳細な使用説明は [packages/cli/README.md](./packages/cli/README.md) を参照。
